Vivado 调用 Vim 作为编辑器

2019-06-14

Vim 作为老牌编辑器，用来编辑 Verilog 和 VHDL 再合适不过了，至少比 Vivado 自带的编辑器好用，而且如果已经习惯了 Vim，所有代码都可以用 Vim 来编辑，岂不快哉。
下面就开始介绍怎么设置在 Vivado 中设置 Vim 作为默认编辑器，这样双击代码就可以直接用 Vim 打开。

安装 Vim

Vim 最好用的版本还是自己的版本，随时添加各种快捷键和插件，找一个开发好的版本然后不断的添加想要的插件和快捷键就好，注意如果插件太多的话，最好有个固态硬盘支持，不然每次开机第一次打开 Vim 都会很慢。

记住 Vim 安装的路径，也就是 "gvim.exe" 这个文件的所在位置。

在 Vivado 中设置 Vim 作为编辑器

方法一 （成功）

  • 在 Vivado 中点击 Tools --> Settings
  • 弹出对话框中点击 Text Editor
  • 然后在 Current Editor 下拉框中选择 Custom Editor
  • 点击后面的三个小点，在弹出的对话框中填入 gvim 路径 [file name] +[line number]

比如我的路径是 D:/Program Files/vim/vim81/gvim.exe [file name] +[line number]
注意 + 后面不能有空格
**

方法二 未成功

在 Vavado 中内置了 Vim 的这个选项，但是前提是把 Vim 的路径添加到环境变量中，但是我添加了一直没有效果，不知道为什么，后面有时间了再找问题。
